# Calculator

Calculator is a simple web application that allows users to perform basic arithmetic operations with two numbers.  
The app supports addition, subtraction, multiplication, and division.

## Demo

Live demo: https://calculator-app-dev.netlify.app/

## Features

- Enter two numbers
- Choose an arithmetic operation
- Calculate the result
- Display the result on the page
- Change result color depending on the value
- Simple responsive layout using Bootstrap

## Technologies

- HTML
- JavaScript
- Bootstrap

## How to View the Project

You can view the project online using the live demo link above.

To run it locally, download or clone the repository and open the `index.html` file in your browser.

## About the Project

This project was created to practice JavaScript basics, working with DOM elements, event handling, functions, conditions, and simple arithmetic operations.
